Simple way to integrate AWeber mailing lists to Contact Form 7. Save your subscribers in a proper mailing list.

== Description ==

Integrate Contact Form 7 with [AWeber](http://renzojohnson.com/contributions/contact-form-7-AWeber-extension). Automatically add form submissions to predetermined lists in AWeber, using its latest API. This AWeber Extension supports multiple mailing lists and API Keys.


= Key Features =

* Easy to use
* Use a different mailing list per contact form
* Single opt-in - no confirmation email send to subscribers
* Double opt-in - confirmation email send to subscribers
* [Opt-in checkbox](http://renzojohnson.com/contributions/contact-form-7-AWeber-extension/AWeber-opt-in-checkbox) - add the chance to opt-in/opt-out
* Support for [custom fields](http://renzojohnson.com/contributions/contact-form-7-AWeber-extension/AWeber-custom-fields)
* Latest AWeber API
* Constantly updated

== Installation ==

1. Upload the entire '[contact-form-7-mail-awb-extension](http://renzojohnson.com/contributions/contact-form-7-AWeber-extension)' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find '[AWeber: Subscriber List Details](http://renzojohnson.com/contributions/contact-form-7-AWeber-extension)' menu in your Contact Form 7 editor.

For basic usage, you can also have a look at the [Contact Form 7 AWeber Extension homepage](http://renzojohnson.com/contributions/contact-form-7-AWeber-extension), you can also have a look at the [Contact Form 7 homepage](http://contactform7.com).
